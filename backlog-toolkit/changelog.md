# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-01-27

### Added
- LICENSE file (MIT License) for clear legal usage rights
- VERSION file for version tracking
- changelog.md following Keep a Changelog format
- examples directory with filled-out template examples
- template-catalog.md with quick reference table and use case matrix
- quick-reference.md with cheat sheets for status, priority, and story points
- config.yaml for project customization
- glossary.md with Agile/Scrum and backlog terminology
- Template validation scripts in scripts directory
- Enhanced README with Table of Contents, Prerequisites, Features, FAQ, and Troubleshooting sections
- Template metadata headers with version and compatibility info
- Validation checklists at end of templates
- Placeholder/helper text in templates with examples
- Process diagrams using Mermaid
- Cross-reference fixes throughout all documentation

### Changed
- Updated all cross-references to use relative paths consistently
- Enhanced README with comprehensive sections
- Improved template documentation with better examples and guidance
- Updated process documentation with visual diagrams

### Fixed
- Fixed inconsistent file path references in templates
- Standardized all cross-references to relative paths
- Removed external dependencies in README
